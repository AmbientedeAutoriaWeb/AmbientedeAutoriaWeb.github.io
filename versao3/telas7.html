Interfaces Terceira Versão

Ajuda

A parte do ambiente Ajuda é responsável por fornecer auxilio aos usuários para utilizarem o ambiente.

A Figura 1. apresenta a ajuda para o perfil professor. Nesta tela o professor deve escolher qual é a parte do ambiente que deseja ajuda.


Após o professor escolher uma parte do ambiente que deseja ajuda, o ambiente lista as funcionalidades referentes a parte escolhida, como na Figura 2.


A Figura 3. apresenta a tela de ajuda para a funcionalidade escolhida pelo professor. Nesta tela, além das informações escritas, o ambiente apresenta imagens para explicar o funcionamento do ambiente.


A Figura 4. apresenta a ajuda para o perfil aluno.


A Figura 5. apresenta a lista de funcionalidades para a parte escolhida pelo aluno.


A partir desta versão, além do texto escrito, o ambiente conta com figuras para auxiliar o aluno a entender o funcionamento do ambiente, comona Figura 6.

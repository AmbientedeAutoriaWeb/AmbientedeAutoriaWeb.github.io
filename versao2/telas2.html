Interfaces Segunda Versão

Criar Projeto

Com a funcionalidade Criar Projeto, o professor pode elaborar os seus projetos de estudos para serem trabalhados com os alunos. Esta funcionalidade é disponível somente para o perfil professor. A seguir são apresentadas as telas que compõem está funcionalidade.

A Figura 1. mostra o formulário com os dados a serem preenchidos para a criação de um projeto de estudo. Este formulário é composto pelos campos Nome, Descrição, Área do Conhecimento e Escola. Os campos do formulário marcados com asterisco '*' são de preenchimento obrigatório. Nesta tela o professor, também, pode interagir com o menu horizontal e com a barra de pesquisa de projetos.


Ao criar um projeto o professor deve associar conteúdos a ele. Para isso, o professor deve selecionar uma área do conhecimento. Como na Figura 2., área do conhecimento 'História' esta selecionada e o ambiente apresentou a lista de conteúdos relacionados a esta área em uma caixa de diálgo. O professor deve marcar os conteúdos que deseja e pressionar o botão "Incluir Conteúdos".


Para que o professor possa aplicar o projeto criado nas turmas que deseja é necessário associar turmas ao projeto. Para realizar esta ação o professor deve selecionar uma escola para que o ambiente apresente uma lista com as turmas relacionadas a esta escola em uma caixa de diálgo, como na Figura 3. O professor deve marcar as turmas que deseja aplicar o projeto e pressionar o botão "Incluir Turmas".


Após o professor preencher o formulário de criação de projeto corretamente e pressionar o botão "Salvar Projeto", o ambiente cria o projeto e exibe uma mensagem de alerta como na Figura 4.


Caso o professor não preencha os campos obrigatórios ao criar um projeto e pressione o botão Salvar Projeto, o ambiente apresenta na tela uma mensagem de alerta e destaca os campos obrigatórios, como na Figura 5.


Após o professor selecionar os conteúdos que deseja adicionar ao projeto e pressicionar o botão "Incluir Conteúdos", o ambiente apresenta uma mensagem de confirmação da ação, como na Figura 6.


Ao incluir turmas ao projeto, o ambiente apresenta uma mensagem de confirmação da ação, como na Figura 7.


Meus Projetos

A funcionalidade Meus Projetos é responsável por listar os projetos relacionados ao usuário autenticado no ambiente. A seguir são apresentadas as telas relacionadas a esta funcionalidade.

A Figura 8. apresenta a lista de projetos relacionados ao professor autenticado no ambiente. Nesta interface o professor pode visualizar todos os projetos relacionados a ele, excluir os projetos e abrir os projetos. Nesta versão esta tela é a página incial do ambiente para o perfil professor


Para navegar pelos projetos relacionados ao professor autenticado no ambiente, também, é possível utilizar os filtros Escola e Turmas, como na Figura 9. O ambiente só exibe o filtro Turma após o professor selecionar uma escola no campo Escola.


Caso o professor selecione opções nos filtros que não tenham nenhum projeto relacionado, o ambiente apresenta uma mensagem como na Figura 10.


A Figura 11. exibe a lista de projetos relacionados ao aluno autenticado no ambiente de autoria Web.


Editar Projeto

A funcionalidade Editar Projeto tem o objetivo de permitir que os professores editem os projetos criados por eles. A seguir são apresentados as telas referentes a esta funcionalidade.

A Figura 12 mostra a interface do projeto com os campos Nome e Descrição do mesmo preenchidos pelo professor ao criar o projeto. Este formulário, também, apresenta o campo Avaliação para que o professor possa avaliar o projeto. Nesta interface o professor pode interagir com os menus horizontais. O menu horizontal do projeto é composto pelos itens: Unidades, Dados do Proejto, Conteúdos, Turmas, Compartilhar Autoria e Publicar Projto.


Caso o professor edite os dados do projeto e deixe os campos obrigatórios em branco o ambiente apresenta uma mensagem de alerta e destaca os campos, como na Figura 13.


A Figura 14. exibe a tela com os conteúdos associados ao projeto. Nesta interface o professor pode excluir os conteúdos associados ao projeto e adicionar novos conteúdos.


Para o professor associar novos conteúdos ao projeto é necessário escolher uma área do conhecimento para que o ambiente liste os conteúdos relacionados a esta área, como na Figura 15.


O ambiente apresenta uma mensagem de confimação com a lista de conteúdos marcados para que o professor confirme a inclusão dos projetos, como na Figura 16.


Quando o professor excluir um conteúdo associado ao projeto o ambiente exibe uma mensagem de alerta, como na Figura 17.


O professor pode visualizar as turmas associadas ao projeto, como na Figura 18. Nesta tela o professor pode adicionar novas turmas ou remover.


Caso o professor deseje associar novas turmas ao projeto é preciso primeiramente selecionar uma escola no campo Escola para que o ambiente liste as turmas que o professor ministra aula na escola seleciona, como na Figura 19.


A Figura 20. exibe a mensagme de confirmação para inclusão de turmas ao projeto.


A Figura 21. apresenta a mensagem de alerta apresentada pelo ambiente quando o professor excluir uma turma relacionada ao projeto.


Excluir Projeto

O ambiente permite que o usuário exclua os projetos associados a ele. A funcionalidade responsável por isso é a Excluir Projeto. Esta funcionalidade é disponível somente para o perfil professor. A seguir são mostradas as telas desta funcionalidade.

Caso o professor deseja excluir um projeto, ele deve ir até a tela com a lista de projetos relacionados a ele e pressionar o botão X. O ambiente exibirá uma mensagem de confirmação da exclusão do projeto para evitar que o professor cometa uma ação indesejada, como na Figura 22.


A Figura 23. exibe a mensagem que confirma a exclusão do projeto, caso o professor aprove a ação de excluir o projeto.


Compartilhar Autoria

A funcionalidade Compartilhar Autoria do projeto permite que o professor compartilhe seus projetos com outros professores. Ao compartilhar um projeto, o professor possibilita que outros professores editem o seu projeto, tornando assim a autoria do projeto compartilhada. A seguir são apresentadas as interfaces referentes a esta funcionalidade.

A Figura 24. apresenta a tela de compartilhamento do projeto. Para buscar o professor com que deseja-se compartilhar o projeto, basta selecionar a escola onde este professor leciona no campo Escola.


Ao selecionar uma escola o ambiente apresenta uma lista de professores relacionados a ela, como na Figura 25. Para compartilhar o projeto basta o professor marcar os professores com quem deseja compartilhar e pressionar o botão Compartilhar Autoria.


A Figura 26. apresenta a mensagem de confirmação do compartilhamento do projeto.


Para compartilhar o projeto, antes o professor precisa confirmar a ação, como na Figura 27.


Publicar Projeto

O ambiente de Autoria Web permite que o professor publique o projeto para professores de outras escolas possam aplica-lo. A funcionalidade responsável por isso é a Publicar Projeto. A seguir são apresentadas as telas referentes a esta funcionalidade.

Para publicar um projeto o professor deve pressionar o item de menu Publicar. O ambiente irá publicar o projeto e exibir uma mensagem de alerta como na Figura 28.


Contudo, para publicar um projeto é necessário avalia-lo antes. A Figura 29. exibe a tela Dados do Projeto com o campo Avaliação e a mensagem de alerta orientando o professor a avaliar o projeto antes de publica-lo, caso ainda não tenha feito.


Caso o professor não preencha o campo Avaliação e pressione Publicar Projeto o ambiente destaca o campo Avaliação e exibe uma mensagem de alerta para o usuário, como na Figura 30.


Pesquisar Projeto

A funcionalidade Pesquisar Projeto permite que o professor pesquise todos os projetos cadastrados no ambiente. A seguir são apresentadas as telas referentes a esta funcionalidade.

Para pesquisar um projeto cadastrado no ambiente o professor precisa utilizar a barra de pesquisa, como na Figura 31. Como pode ser visto nesta figura nesta versão os resultados da pesquisa são exibidos separadamente.


Caso a pesquisa do realizada não retorne nenhum projeto, o ambiente exibe uma mensagem de alerta, como na Figura 32.


Ao digitar uma informação com erro de ortografia no campo Pesquisa, o ambiente apresenta uma mensagem, como na Figura 33.


Utilizar Projeto

A funcionalidade Utilizar Projeto permite que o professor aplique em suas turmas os projetos publicados por outros professores. A seguir são apresentadas as interfaces referentes a esta funcionalidade.

Ao realizar uma pesquisa o professor pode selecionar um projeto para trabalhar com suas turmas. Para utilizar um projeto publicado por outro professor, basta associar turmas a este projeto. A Figura 34. mostra a tela com os dados do projeto e uma área para o professor associar as turmas que deseja aplicar o projeto.


Ao selecionar uma escola, o ambiente apresenta uma lista contendo todas as turmas da escola que o professor autenticado no ambiente leciona, como na Figura 35.


A Figura 36. exibe as turmas associadas ao projeto.


Listar Projetos Publicados

A funcionalidade Listar Projetos Publicados, tem a função de listar projetos recentemente publicados pelos professores cadastrados no ambiente. A seguir é apresentado a tela referente a esta funcionaliade.

A Figura 37. exibe a lista de projetos publicados pelos professores cadastrados no ambiente. Nesta lista o professor pode escolher um projeto que deseja utilizar em sua sala de aula.

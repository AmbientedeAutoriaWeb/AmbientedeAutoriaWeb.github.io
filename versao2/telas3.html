Interfaces Segunda Versão

Criar Unidade

O ambiente permite que o professor crie unidades para trabalhar com os alunos. A funcionalidade Criar Unidade é responsável por esta tarefa. A seguir são apresentadas as telas referentes a esta funcionalidade.

A Figura 1. apresenta o formulário para a criação da unidade. Este formulário conta com somente um único campo, Nome.


Caso o professor pressione o botão Salvar Unidade sem preencher o campo Nome que é obrigatório, o ambiente destaca o campo e exibe uma mensagem de alerta como na Figura 2.


Após o professor preencher o campo Nome e pressionar o botão Salvar Unidade, o ambiente apresenta uma mensagem de alerta para confirmar a criação da unidade, como na Figura 3.


Editar Unidade

A funcionalidade Editar Unidade tem a tarefa de permitir que o professor edite os dados da unidade. A seguir são exibidas as telas relacionadas a esta funcionalidade.

A Figura 4. exibe o formulário preenchido com os dados da unidade. Neste formulário o professor pode alterar os dados da unidade e salvar.


Caso o professor deixe o campo Nome em branco e pressiona o botão Salvar Unidade, o ambiente destaca o campo Nome e exibe uma mensagem de alerta ao usuário como na Figura 5.


Excluir Unidade

O ambiente contem a funcionalidade Excluir Unidade, esta funcionalidade permite que o professor exclua as unidades relacionadas ao projeto. A seguir são exibidas as interfaces referentes a esta funcionalidade.

Para excluir uma unidade o professor deve pressionar o botão X relacionado a unidade. A Figura 6. exibe a mensagem de confirmação da ação de excluir uma unidade. Esta mensagem é exibida para que o professor não efetue uma ação indesejada.


A Figura 7. exibe a mensagem confirmação da exclusão da unidade.


Listar Unidade

A funcionalidade Listar Unidade é responsável por listar as unidades do projeto que esta aberto no ambiente. Esta funcionalidade é disponível tanto para o perfil professor quando para o perfil aluno. A seguir são apresentadas as telas relacionadas a esta funcionalidade.

A Figura 8. apresenta a lista de unidades para o perfil aluno. Nesta interface o aluno pode interagir também com o menu horizontal e com a barra de pesquisa.


A lista de unidades apresentada pelo ambiente para o perfil professor é exibida na Figura 9. Nesta interface o professor pode editar e excluir uma unidade e interagir com os menus horizontais e com a barra de pesqiusa.

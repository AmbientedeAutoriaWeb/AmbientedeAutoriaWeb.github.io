Esta página Web apresenta as três versões dos casos de uso e das interfaces do ambiente de Autoria Web. Além disso, esta página Web apresenta uma comparação das três versões da interface do ambiente.

A primeira versão do ambiente de Autoria Web foi submetida a uma avaliação heurística. A partir dos dados coletados nesta avaliação foram realizadas melhorias no ambiente, criando assim a segunda versão dos casos de uso e das interfaces do ambiente.

A segunda versão do ambiente foi submetida a um ensaio de interação com professores do Ensino Fundamental (1º ao 5º ano) da rede municipal de Curitiba. A partir da análise dos dados coletados no ensaio de interação foi criada a terceira versão dos casos de uso e das interfaces do ambiente.

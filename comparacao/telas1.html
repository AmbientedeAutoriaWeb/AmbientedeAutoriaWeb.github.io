Interfaces Comparação das Versões

Acessar ambiente

A funcionalidade Acessar Ambiente é responsável pelo controle de acesso ao ambiente. Nesta funcionalidade é realizada a autenticação do usuário através do login e da senha. A seguir são apresentadas a evolução das telas desta funcionalidade.


1ª Versão


A tela exibida na Figura 1. representa a tela inicial do ambiente onde o professor e o aluno informam o seus respectivos login para acessar o ambiente. Está tela consiste em duas partes: a área do professor e a área do aluno. A área do professor é composta pelo campo CPF e pelo botão Pesquisar que devem ser utilizados para acessar o ambiente. Já a área do aluno é composta pelos campos Nome e Data de Nascimento e pelo botão Pesquisar que também devem ser utilizados para acessar o ambiente.


Após o usuário preencher os campos da Figura 1. e pressionar o botão Pesquisar, o ambiente apresenta o campo Senha, o link Esqueci minha Senha e o botão Entrar, como na Figura 2.

2ª e 3ª Versão


A partir da segunda versão a tela exibida na Figura 3. representa a tela inicial do ambiente. Nesta versão não há mais a função pesquisar por usuário, agora o professor e o aluno informam os dados de acesso em um único passo por sugestão dos especialistas durante a avaliação heurística. Nesta tela o usuário pode já informar a sua senha ou utilizar o link Esqueci minha senha. Para realizar o cadastro no ambiente, a partir da segunda versão, o usuário deve utilizar o link Cadastra-se no ambiente. Além disso, foi adicionado uma mascara no campo CPF para o perfil professor, como requisitado durante a avaliação heurística.

1ª Versão

A Figura 4. traz como era a mensagem de alerta do ambiente na primeira versão das telas. Nesta figura é apresentado a mensagem de alerta para quando o professor não preenche o campo CPF e pressiona o botão Pesquisar.


2ª Versão

As mensagens de alerta para a segunda versão foram modificadas. Nesta versão foram adicionados marcadores semânticos, seguindo as remcondações dos especialistas durante a avaliação heurística, como na Figura 5. Esta figura exibe a mensagem de alerta para quando o professor não preenche o campo CPF e pressiona o campo Entrar.


3ª Versão

Para a versão três das interfaces do ambiente, as mensagens de alerta também foram alteradas, como na Figura 6. Esta modificação foi feita porque alguns professores tiveram dificuldade em interagir com as mensagens de alerta durante o teste de usabilidade.


Primeiro Acesso

A primeira vez que um professor ou a aluno acessa o ambiente é necessário realizar o cadastro de uma senha, sendo está a tarefa da funcionalidade Primeiro Acesso. Com está funcionalidade o professor e o aluno podem realizar o cadastro de uma senha para utilizar o ambiente proposto. As comparação das telas que compõem a funcionalidade Primeiro acesso estão representadas a seguir.

1ª Versão

A Figura 7. representa o formulário que deve ser preenchido para salvar uma senha no ambiente. Está tela é composta pelos campos Senha e Confirmar Senha, além do botão Salvar e Entrar. Na primeira versão esta tela parece se o ambiente não encontra o cadastro da senha do usuário.


2ª e 3ª Versão

A partir da segunda versão para o usuário cadastrar uma senha no ambiente é necessário clicar no link Cadastrar-se no ambiente. A Figura 8. a presenta o formulário para o usuário escolher em qual perfil deseja realizar o cadastro.


A Figura 9. apresenta o formulário de cadastro de senha no ambiente para o perfil professor.


Já a Figura 10. apresenta o formulário de cadastro de senha no ambiente para o perfil aluno.


Esqueci a Senha

A funcionalidade Esqueci a Senha tem a tarefa de cadastrar uma nova senha para o usuário, caso o mesmo tenha esquecido a sua senha de acesso ao ambiente. Assim, a seguir são apresentadas a comparação das telas dessa funcionalidade.

1ª Versão

A Figura 11. apresenta o formulário a ser preenchido para cadastrar uma nova senha no ambiente, caso o usuário a tenha esquecido. Além disso, esta tela apresenta os dados do usuário: nome, data de nascimento e o perfil do usuário.


2ª e 3ª Versão

Como a tela de acesso ao usuário foi alterada na segunda versão, a funcionalidade Esqueci minha senha também precisou ser alterada. A Figura 12. apresenta o formulário de requisição de uma nova senha de acesso ao ambiente para o perfil professor.


Já a Figura 13. apresenta o formulário de requisição de uma nova senha para o perfil aluno.


Alterar Senha

O ambiente proposto nesta dissertação permite que os usuários alterem a senha de acesso ao ambiente. Assim, a funcionalidade Alterar Senha permite modificar a senha do usuário cadastrada no ambiente. A seguir é apresentado a comparação das telas desta funcionalidade.

1ª Versão

A Figura 14. apresenta a tela com o formulário para o usuário alterar a senha de acesso ao ambiente. Este formulário é composto pelos campos: Senha, Nova Senha e Confirmar Nova Senha. Ademais, nesta tela o usuário pode interagir com o menu horizontal.


2ª e 3ª Versão

A Figura 15. apresenta o formulário de alteração de senha para a segunda e terceira versão das telas. Nesta figura pode ser visto a alteração no menu horizontal e as opções Alterar Senha, Ajuda e Sair que ficam, a partir da segunda versão, no topo direito da tela.

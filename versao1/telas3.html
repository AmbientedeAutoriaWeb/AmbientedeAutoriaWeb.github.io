Criar Atividade

O ambiente permite que o professor crie atividades para trabalhar com os alunos. A funcionalidade Criar Atividade é responsável por esta tarefa. A seguir são apresentadas as telas referentes a esta funcionalidade.

A Figura 1. apresenta o formulário para a criação da atividade. Este formulário conta com somente um único campo, Nome.


Caso o professor pressione o botão Salvar Atividade sem preencher o campo Nome que é obrigatório, o ambiente destaca o campo e exibe uma mensagem de alerta como na Figura 2.


Após o professor preencher o campo Nome e pressionar o botão Salvar Atividade, o ambiente apresenta uma mensagem de alerta para confirmar a criação da atividade, como na Figura 3.


Editar Atividade

A funcionalidade Editar Atividade tem a tarefa de permitir que o professor edite os dados da atividade. A seguir são exibidas as telas relacionadas a esta funcionalidade.

A Figura 4. exibe o formulário preenchido com os dados da atividade. Neste formulário o professor pode alterar os dados da atividade e salvar.


Caso o professor deixe o campo Nome em branco e pressiona o botão Salvar Atividade, o ambiente desta o campo Nome e exibe uma mensagem de alerta ao usuário como na Figura 5.


Excluir Atividade

O ambiente contem a funcionalidade Excluir Atividade, esta atividade permite que o professor exclua as atividades relacionadas ao projeto. A seguir são exibidas as interfaces referentes a esta funcionalidade.

Para excluir uma atividade o professor deve pressionar o botão X relacionado a atividade. A Figura 6. exibe a mensagem de confirmação da ação de excluir uma atividade. Esta mensagem é exibida para que o professor não efetue uma ação indesejada.


A Figura 7. exibe a mensagem confirmação da exclusão da atividade.


Listar Atividade

A funcionalidade Listar Atividade é responsável por listar as atividades do projeto que esta aberto no ambiente. Esta funcionalidade é disponível tanto para o perfil professor quando para o perfil aluno. A seguir são apresentadas as telas relacionadas a esta funcionalidade.

A Figura 8. apresenta a lista de atividades para o perfil aluno. Nesta interface o aluno pode interagir também com o menu horizontal e utilizar a paginação para navegar pelas atividades.


A lista de atividades apresentada pelo ambiente para o perfil professor é exibida na Figura 9. Nesta interface o professor pode editar e excluir uma atividade e interagir com o menu horizontal e vertical, além de utilizar a paginação para navegar pelas atividades.

Interfaces Primeira Versão

Criar Projeto

Com a funcionalidade Criar Projeto, o professor pode elaborar os seus projetos de estudos para serem trabalhados com os alunos. Esta funcionalidade é disponível somente para o perfil professor. A seguir são apresentadas as telas que compõem está funcionalidade.

Para criar um projeto no ambiente de autoria Web é necessário acessar o menu horizontal do ambiente, apresentado na Figura 1. Este menu é composto pelos itens: Criar Projeto, Listar Projeto, Nova Senha, Ajuda e Sair. Para criar um projeto o professor deve pressionar o item de menu Criar Projeto.


A Figura 2. mostra o formulário com os dados a serem preenchidos para a criação de um projeto de estudo. Este formulário é composto pelos campos Nome, Descrição, Área do Conhecimento e Escola. Os campos do formulário marcados com asterisco '*' são de preenchimento obrigatório. Para que o professor saiba onde clicou no menu horizontal o item 'Projeto' esta destacado. Nesta tela o professor, também, pode interagir com o menu horizontal e com a barra de pesquisa de projetos.


Ao criar um projeto o professor deve associar conteúdos a ele. Para isso, o professor deve selecionar uma área do conhecimento. Como na Figura 3, área do conhecimento 'História' esta selecionada e o ambiente apresentou a lista de conteúdos relacionados a esta área. Caso a área do conhecimento tenha mais de dez conteúdos relacionados a ela, o ambiente apresenta uma paginação para o professor navegar pelos conteúdos. Para associar conteúdos ao projeto, o professor deve marcar os conteúdos que deseja e pressionar o botão Incluir Conteúdos. Nesta tela o professor, também, pode visualizar a lista de conteúdos já associados ao projeto e desfazer esta associação.


Para que o professor possa aplicar o projeto criado nas turmas que deseja é necessário associar turmas ao projeto. Para realizar esta ação o professor deve selecionar uma escola para que o ambiente apresente uma lista com as turmas relacionadas a esta escola, como na Figura 4. O professor deve marcar as turmas que deseja aplicar o projeto e pressionar o botão Incluir Turma. Além disso, nesta tela o professor também pode visualizar a lista de turmas já associadas ao projeto e pode desassociar essas turmas.


Caso o professor não preencha ao menos os cmapos obrigátorios do formulário e pressione o botão Salvar projeto, o ambiente apresenta uma mesnagem de alerta como na Figura 5.


Após o professor preencher o formulário de criação de projeto corretamente e pressionar o botão Salvar Projeto, o ambiente cria o projeto e exibe uma mensagem de alerta como na Figura 6.


A Figura 7 apresenta a mensagem de alerta para o caso de professor pressionar o botão Incluir Conteúdos sem ter selecionado nenhum conteúdo.


Ao pressionar o botão Incluir Turma o professor deve ter selecionado ao menos uma turma na lista de turmas. Caso o professor não execute esta ação, o ambiente destaca a lista de turmas e apresenta uma mensagem de alerta, como na Figura 8.


Listar Projeto

A funcionalidade Listar Projeto é responsável por listar os projetos relacionados ao usuário autenticado no ambiente. A seguir são apresentadas as telas relacionadas a esta funcionalidade.

A Figura 9. apresenta a lista de projetos relacionados ao professor autenticado no ambiente. Nesta interface o professor pode visualizar todos os projetos relacionados a ele utilizando a paginação, excluir os projetos e abrir os projetos.


Para navegar pelos projetos relacionados ao professor autenticado no ambiente, também, é possível utilizar os filtros Escola e Turmas, como na Figura 10. O ambiente só exibe o filtro Turma após o professor selecionar uma escola no campo Escola.


Caso o professor selecione opções nos filtros que não tenham nenhum projeto relacionado, o ambiente apresenta uma mensagem como na Figura 11.


A Figura 12. exibe a lista de projetos relacionados ao aluno autenticado no ambiente de autoria Web.


Editar Projeto

A funcionalidade Editar Projeto tem o objetivo de permitir que os professores editem os projetos criados por eles. A seguir são apresentados as telas referentes a esta funcionalidade.

A Figura 13 mostra a interface do projeto com os campos Nome e Descrição do mesmo preenchidos pelo professor ao criar o projeto. Este formulário, também, apresenta o campo Avaliação para que o professor possa avaliar o projeto. Nesta interface o professor pode interagir com o menu horizontal e com o menu vertical. O menu vertical é composto pelos itens: Atividades, Dados do Proejto, Conteúdos, Turmas, Compartilhar e Publicar.


Caso o professor edite os dados do projeto e deixe os campos obrigatórios em branco o ambiente apresenta uma mensagem de alerta e destaca os campos, como na Figura 14.


A Figura 15. exibe a tela com os conteúdos associados ao projeto. Nesta interface o professor pode excluir os conteúdos associados ao projeto e adicionar novos conteúdos. Para que o professor saiba onde encontra-se o item de menu Conteúdos fica destacado dos demais.


Para o professor associar novos conteúdos ao projeto é necessário escolher uma área do conhecimento para que o ambiente liste os conteúdos relacionados a esta área, como na Figura 16. Nesta interface também é possível visualizar os conteúdos já associados ao projeto.


Após o professor salvar as alterações realizadas nos conteúdos do projeto o ambiente apresenta uma mensagem de alerta e redireciona o professor a lista de atividades associadas ao projeto, como na Figura 17.


Caso o professor não selecione nenhum conteúdo para associar ao projeto e pressione o botão Incluir Conteúdo o ambiente destaca a lista de conteúdos e exibe uma mensagem de alerta, como na Figura 18.


O projeto deve obrigatoriamente ter ao menos um conteúdo relacionado a ele. Então caso o professor tente excluir o único conteúdo associado ao projeto o ambiente apresenta uma mensagem de alerta e destaca o campo Área do Conhecimento, como na Figura 19.


Quando o professor excluir um conteúdo associado ao projeto o ambiente exibe uma mensagem de alerta, como na Figura 20.


O professor pode visualizar as turmas associadas ao projeto, como na Figura 21. Nesta tela o professor pode adicionar novas turmas ou remover. Para que o professor saiba onde está o item de menu Turmas esta destacado.


Caso o professor deseje associar novas turmas ao projeto é preciso primeiramente selecionar uma escola no campo Escola para que o ambiente liste as turmas que o professor ministra aula na escola seleciona, como na Figura 22.


Após o professor pressionar o botão Salvar Turmas o ambiente redireciona o professor para a tela das atividades do projeto e apresenta uma mensagem de alerta, como na Figura 23.


Caso o professor pressione o botão Incluir Turmas sem ter selecionado nenhuma turma na lista, o ambiente destaca a lista de turmas e uma mensagem de alerta, como na Figura 24.


A Figura 25. apresenta a mensagem de alerta apresentada pelo ambiente quando o professor excluir uma turma relacionada ao projeto.


Excluir Projeto

O ambiente permite que o usuário exclua os projetos associados a ele. A funcionalidade responsável por isso é a Excluir Projeto. Esta funcionalidade é disponível somente para o perfil professor. A seguir são mostradas as telas desta funcionalidade.

Caso o professor deseja excluir um projeto, ele deve ir até a tela com a lista de projetos relacionados a ele e pressionar o botão X. O ambiente exibirá uma mensagem de confirmação da exclusão do projeto para evitar que o professor cometa uma ação indesejada, como na Figura 26.


A Figura 27. exibe a mensagem que confirma a exclusão do projeto, caso o professor aprove a ação de excluir o projeto.


Compartilhar Projeto

A funcionalidade Compartilhar Projeto permite que o professor compartilhe seus projetos com outros professores. Ao compartilhar um projeto, o professor possibilita que outros professores editem o seu projeto, tornando assim a autoria do projeto compartilhada. A seguir são apresentadas as interfaces referentes a esta funcionalidade.

A Figura 28. apresenta a tela de compartilhamento do projeto. O item do menu Compartilhar encontra-se destacado dos demais para indicar ao usuário onde ele está. Para buscar o professor com que deseja-se compartilhar o projeto, basta selecionar a escola onde este professor leciona no campo Escola.


Ao selecionar uma escola o ambiente apresenta uma lista de professores relacionados a ela, como na Figura 29. Para compartilhar o projeto basta o professor marcar os professores com quem deseja compartilhar e pressionar o botão Compartilhar Projeto.


A Figura 30. apresenta a mensagem de confirmação do compartilhamento do projeto.


Caso o professor pressione o botão Compartilhar Projeto sem ter marcado os professores com quem deseja compartilhar o projeto, o ambiente apresenta uma mensagem de alerta e destaca a lista de professores, como na Figura 31.


Publicar Projeto

O ambiente de Autoria Web permite que o professor publique o projeto para professores de outras escolas possam aplica-lo. A funcionalidade responsável por isso é a Publicar Projeto. A seguir são apresentadas as telas referentes a esta funcionalidade.

Para publicar um projeto o professor deve pressionar o item de menu Publicar. O ambiente irá publicar o projeto e exibir uma mensagem de alerta como na Figura 32.


Contudo, para publicar um projeto é necessário avalia-lo antes. A Figura 33. exibe a tela Dados do Projeto com o campo Avaliação e a mensagem de alerta orientando o professor a avaliar o projeto antes de publica-lo, caso ainda não tenha feito.


Caso o professor não preencha o campo Avaliação e pressione Publicar Projeto o ambiente destaca o campo Avaliação e exibe uma mensagem de alerta para o usuário, como na Figura 34.


Pesquisar Projeto

A funcionalidade Pesquisar Projeto permite que o professor pesquise todos os projetos cadastrados no ambiente. A seguir são apresentadas as telas referentes a esta funcionalidade.

Para pesquisar um projeto cadastrado no ambiente o professor precisa utilizar a barra de pesquisa, como na Figura 35.


Caso a pesquisa do realizada não retorne nenhum projeto, o ambiente exibe uma mensagem de alerta, como na Figura 36.


Ao digitar uma informação com erro de ortografia no campo Pesquisa, o ambiente apresenta uma mensagem, como na Figura 37.


Utilizar Projeto

A funcionalidade Utilizar Projeto permite que o professor aplique em suas turmas os projetos publicados por outros professores. A seguir são apresentadas as interfaces referentes a esta funcionalidade.

Ao realizar uma pesquisa o professor pode selecionar um projeto para trabalhar com suas turmas. Para utilizar um projeto publicado por outro professor, basta associar turmas a este projeto. A Figura 38. mostra a tela com os dados do projeto e uma área para o professor associar as turmas que deseja aplicar o projeto.


Ao selecionar uma escola, o ambiente apresenta uma lista contendo todas as turmas da escola que o professor autenticado no ambiente leciona, como na Figura 39.


A Figura 40. exibe as turmas associadas ao projeto.

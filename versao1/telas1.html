Interfaces Primeira Versão

Acessar ambiente

A funcionalidade Acessar Ambiente é responsável pelo controle de acesso ao ambiente. Nesta funcionalidade é realizada a autenticação do usuário através do login e da senha. A seguir são apresentadas as telas que compõem está funcionalidade.


A tela exibida na Figura 1. representa a tela inicial do ambiente onde o professor e o aluno informam o seus respectivos login para acessar o ambiente. Está tela consiste em duas partes: a área do professor e a área do aluno. A área do professor é composta pelo campo CPF e pelo botão Pesquisar que devem ser utilizados para acessar o ambiente. Já a área do aluno é composta pelos campos Nome e Data de Nascimento e pelo botão Pesquisar que também devem ser utilizados para acessar o ambiente.


Para realizar o acesso ao ambiente, tanto o professor quanto o aluno devem informar os dados requisitados na tela e pressionar o botão Pesquisar para que o ambiente verifique a autenticidade dos dados informados.


A Figura 2. traz a área do professor após o ambiente verificar a veracidade do CPF informado. A tela apresenta o campo Senha que deve ser informado antes de pressionar o botão Entrar, para o ambiente verificar a autenticidade da senha. Ademais, a tela traz o link Esqueci minha senha para que o professor realize um novo cadastro de senha caso a tenha esquecido. É importante reassaltar que esta tela não existe na segunda versão do ambiente.


Caso o professor pressione o botão Pesquisar sem ter informado o campo CPF o ambiente deve emitir uma mensagem de alerta e destacar o campo CPF, como na Figura 3.


Para acessar ambiente o professor deve também informar a senha antes de pressionar o botão Entrar, caso isto não aconteça o ambiente apresenta uma mensagem de alerta na interface e destaca o campo Senha de vermelho, como na Figura 4.


A Figura 5. traz a mensagem de alerta apresentada pelo ambiente quando o professor informa a senha incorreta ao tentar acessar o ambiente.


Quando o aluno tenta acessar o ambiente, ele deve informar os campos Nome e Data de Nascimento e pressionar o botão Pesquisar. Caso o ambiente verifique que os dados são validos, ele exibe na tela o campo Senha, o link Esqueci minha senha e o botão Entrar. A Figura 6. apresenta a tela após a validação dos campos Nome e Data de Nascimento.


A Figura 7. traz a mensagem de alerta e o campo Nome e Data de Nascimento em destaque, para quando o aluno pressionar o botão Pesquisar sem informar estes campos.


A mensagem de alerta para quando o aluno tentar acessar o ambiente sem informar a senha é exibida na Figura 8., juntamente com o campo Senha destacado.


Caso o aluno informe a senha incorretamente ao tentar acessar o ambiente é exibida uma mensagem de alerta e o campo Senha é destacado, como na Figura 9.


A mensagem de alerta exibida na Figura 10. é apresentada pelo ambiente quando o professor ou o aluno não tem cadastro no ambiente. Está mensagem é exibida após o professor e o aluno informarem os seus login e pressionarem o botão Pesquisar.


Primeiro Acesso

A primeira vez que um professor ou a aluno acessa o ambiente é necessário realizar o cadastro de uma senha, sendo está a tarefa da funcionalidade Primeiro Acesso. Com está funcionalidade o professor e o aluno podem realizar o cadastro de uma senha para utilizar o ambiente proposto. As telas que compõem a funcionalidade Primeiro acesso estão representadas a seguir, sendo importante destacar que as telas apresentadas comportam tanto o perfil professor quanto o perfil aluno.

A Figura 11. representa o formulário que deve ser preenchido para salvar uma senha no ambiente. Está tela é composta por dois campos Senha e Confirmar Senha além do botão Salvar e Entrar.


Para salvar uma senha no ambiente é necessário preencher todos os campos apresentados na Figura 11. Caso o usuário não informe o campo Senha, o ambiente exibe uma mensagem de alerta em tela e destaca o campo Senha, como na Figura 12.


O campo Confirmar Senha também tem o seu preenchimento obrigatório ao cadastrar uma senha no ambiente. Então caso o usuário não informe o campo Confirmar Senha, o ambiente exibe na tela uma mensagem de alerta e destaca o campo, como na Figura 13.


Na Figura 14. traz a mensagem de alerta e os campos Senha e Confirmar Senha em destaque para quando o usuário não informar a mesma senha para os dois campos.


Esqueci a Senha

A funcionalidade Esqueci a Senha tem a tarefa de cadastrar uma nova senha para o usuário, caso o mesmo tenha esquecido a sua senha de acesso ao ambiente. Assim, a seguir são apresentadas as telas referentes a funcionalidade Esqueci a Senha. Vale destacar que as telas apresentadas para esta funcionalidade pertencem tanto para o perfil professor quanto para o perfil aluno.

A Figura 15 apresenta o formulário a ser preenchido para cadastrar uma nova senha no ambiente, caso o usuário a tenha esquecido. Além disso, esta tela apresenta os dados do usuário: nome, data de nascimento e o perfil do usuário.


Quando o usuário preenche o formulário apresentado na Figura 15. corretamente e pressiona o botão Salvar e Entrar, o ambiente apresenta uma mensagem de alerta confirmando a alteração de senha, como na Figura 16.


Caso o usuário, ao requirir a alteração de senha, não preencha o campo Nova Senha e pressione o botão Salvar e Entrar, o ambiente apresenta uma mensagem de alerta e destaca o campo Nova Senha, como na Figura 17.


Para cadastrar uma nova senha, o usuário é obrigado a preencher o campo Confirmar Nova senha. Caso este campo não seja preenchido o ambiente apresenta uma mensagem de alerta e destaca o campo Confirmar Nova Senha, como na Figura 18.


A Figura 19. exibe a mensagem de alerta apresentado pelo ambiente, caso os campos Nova Senha e Confirmar Nova Senha não confiram.


Alterar Senha

O ambiente proposto nesta dissertação permite que os usuários alterem a senha de acesso ao ambiente. Assim, a funcionalidade Alterar Senha permite modificar a senha do usuário cadastrada no ambiente. A seguir estão apresentadas as telas que compõem a funcionalidade Alterar Senha. Vale ressaltar que estas telas valem para os dois perfis de usuários que o ambiente comporta.

A Figura 20. apresenta a tela com o formulário para o usuário alterar a senha de acesso ao ambiente. Este formulário é composto pelos campos: Senha, Nova Senha e Confirmar Nova Senha. Ademais, nesta tela o usuário pode interagir com o menu horizontal.


Caso o usuário preencha o formulário da Figura 21. corretamente, o ambiente apresenta em tela uma mensagem de confirmação de alteração de senha, como na Figura 22.


Para alterar a senha de acesso ao ambiente, o usuário deve informar a senha atual de acesso. Caso o usuário não informe o campo Senha e pressione o botão Alterar Senha o ambiente apresenta uma mensagem de alerta e destaca o campo, como na Figura 22.


A Figura 23. mostra a mensagem de alerta e o campo Senha destacado. Está situação acontece quando o usuário informa o campo Senha incorretamente e pressiona o botão Alterar Senha.


Para alterar a senha de acesso ao ambiente o usuário deve informar as mesmas informações nos campos Nova Senha e Confirmar Nova Senha. Caso estes campos não sejam os mesmos o ambiente apresenta uma mensagem de alerta e destaca os dois campos, como na Figura 24.


A Figura 25. apresenta a mensagem de alerta e os campos Nova Senha e Confirmar Nova Senha em destaque para o caso de o usuário não informar estes dois campos ao alterar a senha.
